# 📈 Marketing Funnel & Conversion Performance Analysis

A marketing analytics dashboard designed to analyze customer conversion behavior, campaign effectiveness, and funnel performance using the Bank Marketing dataset.

## 🚀 Features

* Funnel Analysis
* Conversion Rate Tracking
* Campaign Performance Analysis
* Customer Segmentation
* Contact Channel Analysis
* Interactive Dashboard

## 📁 Dataset Overview

Dataset: UCI Bank Marketing Dataset

### Key Columns

* age
* job
* marital
* education
* contact
* month
* campaign
* y (Target Variable)

Target Variable:

* yes → Converted Customer
* no → Not Converted

## 📈 KPIs

* Customers Contacted
* Converted Customers
* Conversion Rate
* Average Campaign Contacts

## 📊 Visualizations

* Marketing Funnel
* Conversion Distribution
* Conversion by Contact Type
* Conversion by Job
* Monthly Conversion Trends
* Age Distribution Analysis
* Campaign Performance Analysis

## 🛠️ Technologies Used

* Python
* Pandas
* Plotly
* Streamlit

## 📦 Installation

```bash
pip install -r requirements.txt
```

## ▶️ Run the Dashboard

```bash
streamlit run app.py
```

## 💡 Business Insights

* Customer occupation influences conversion rates.
* Contact channels impact campaign success.
* Excessive campaign contacts may reduce effectiveness.
* Seasonal trends affect conversion performance.
